# 课程作业管理系统 java1253

## 项目概述

课程作业管理系统是一款旨在简化和优化课程作业流程的在线平台。系统采用前后端不分离的架构设计，集成了多种功能模块，以满足教师和学生的日常需求。

## 技术栈

- 后端：Spring Boot
- 前端：Vue + ElementUI
- Node.js
- 运行环境：JDK 1.8 + IDEA + MySQL + Redis + Tomcat 8

## 功能模块

- **用户登录与注册**
  - 学生注册
  - 个人中心
- **公告管理**
- **用户管理**
  - 学生管理
  - 教师管理
- **课程管理**
  - 学生选课管理
  - 课程资源管理
- **作业管理**
  - 作业布置
  - 作业提交
  - 作业评分管理
- **课程评价管理**

## 系统角色

- 学生
- 教师
- 管理员

## 运行环境

- **服务器：**
  - JDK 1.8
  - Tomcat 8
- **数据库：**
  - MySQL
  - Redis
- **开发工具：**
  - IntelliJ IDEA

## 部署步骤

1. 环境准备：确保JDK 1.8、MySQL、Redis、Tomcat 8已安装并正确配置。
2. 数据库构建：根据项目文档创建数据库和表结构。
3. 后端部署：编译后端代码，部署至Tomcat服务器。
4. 前端部署：编译前端代码，部署至Web服务器。

```
项目根目录/
```
